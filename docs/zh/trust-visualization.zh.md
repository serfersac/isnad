# 信任等级可视化 (Trust Tier Visualization)

ISNAD 协议根据验证者的参与度和准确性将其分为三个核心等级。

## 等级进度图
`[VERIFIED]` --(需 ≥3 名审计员)--> `[TRUSTED]` --(需 ≥10 名审计员)--> `[CERTIFIED]`

## 审计员阈值详情

| 信任等级 | 所需审计员数量 | 描述 |
| :--- | :--- | :--- |
| **VERIFIED** | ≥ 0 | 初始级别，资源已通过基础验证。 |
| **TRUSTED** | ≥ 3 | 中级级别，由多名活跃审计员背书。 |
| **CERTIFIED** | ≥ 10 | 最高级别，代表协议的最高共识。 |
